## scrutiny

scrutiny is an R package for testing GRIM and GRIMMER.

## Methods implemented

- [GRIM](/methods/grim/) — tests whether reported means are consistent with integer data
- [GRIMMER](/methods/grimmer/) — extends GRIM to standard deviations
